About This Project

This advanced matrix calculator was developed by Berat Erol Çelik as a demonstration of web development skills using Flask, Python, and modern frontend technologies.

The calculator performs various matrix operations including addition, multiplication, and advanced checks like idempotent, nilpotent, and orthogonal matrices.
